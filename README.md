# rung

## A simple terminal-based Pong game

**rung** is a minimal Pong clone that runs entirely in your terminal.
Built in Rust using ```crossterm```. The main purpose of this project is to learn the Rust programming language.

## Controls

| Key       | Action                      |
|-----------|-----------------------------|
| `W` / `S` | Move left paddle up / down  |
| `↑` / `↓` | Move right paddle up / down |
| `ESC`     | Exit game                   |
| `R`       | Reset after scoring         |

---

### 1. Clone the repository

```bash
git clone https://github.com/gorgoll/rung.git
cd rung
```

### 2. Build and run

```bash
cargo run
```

---

## Requirements

* Rust
* A terminal

---

## Dependencies

* `crossterm` — terminal control & input handling
* `rand` — randomness for ball behavior
